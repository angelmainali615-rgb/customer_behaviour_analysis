## 🚀 Project Workflow

### 1️⃣ Data Loading
- Imported dataset using Pandas
- Checked data structure and data types
- Identified missing values

### 2️⃣ Exploratory Data Analysis (EDA)
- Generated summary statistics
- Performed distribution analysis
- Conducted correlation analysis
- Created visualizations (bar charts, histograms, line charts)

### 3️⃣ Data Cleaning
- Handled missing values
- Removed duplicates
- Standardized column formats
- Created derived columns where necessary

### 4️⃣ SQL Analysis (PostgreSQL)
- Imported cleaned dataset into PostgreSQL
- Executed queries using:
  - SELECT
  - WHERE
  - GROUP BY
  - ORDER BY

- Calculated key business metrics

### 5️⃣ Power BI Dashboard
- Connected Power BI to PostgreSQL
- Created interactive dashboard with:
  - KPI Cards
  - Trend Analysis
  - Category Comparisons
  - Filters and Slicers

### 6️⃣ Reporting & Presentation
- Prepared structured analytical report
- Summarized insights and findings
